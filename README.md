# 🚀 My CS50 Learning Journey

Welcome to my CS50 learning journey! This repository documents my step-by-step progress through Harvard University's CS50: Introduction to Computer Science. Inside, you will find my notes, problem sets, projects, and personal reflections.

This guide is also designed to help anyone who wants to start this course from scratch.

---

## 🛠️ Step-by-Step Registration Guide (How to Start for Free)

If you are new to CS50 and want to get the **Free Verified Harvard Certificate**, follow these exact steps:

1. **Enroll via edX:** Go to [cs50.edx.org](https://edx.org) and click **Enroll**. Choose the **"Audit this course"** option to take it entirely for free.
2. **Create a GitHub Account:** If you don't have one, sign up at [github.com](https://github.com). You will need it to log into the programming environment and submit assignments.
3. **Authorize CS50:** Visit [cs50.me/cs50x](https://cs50.me), log in with your GitHub account, and complete the quick initial student form.
4. **Access the Web IDE:** Open [cs50.dev](https://cs50.dev), log in with GitHub, and you will get a free, ready-to-use cloud version of Visual Studio Code customized for the course.
